# ⚡ Pokédex Web App (React + Vite)

¡Hola! Bienvenido a este proyecto. Esta es una aplicación web interactiva desarrollada para buscar, visualizar y explorar información detallada sobre distintos Pokémon. El objetivo de este proyecto fue crear una interfaz rápida, limpia y responsiva consumiendo datos externos.

## 🛠️ Tecnologías utilizadas
* **Frontend:** React.js
* **Build Tool:** Vite (para un entorno de desarrollo ultrarrápido)
* **Estilos:** Tailwind CSS
* **Lenguaje:** JavaScript (ES6+)

## 🚀 Características principales
* Búsqueda en tiempo real por Nombre o ID.
* Navegación fluida entre Pokémon utilizando botones de "Siguiente" y "Anterior".
* Visualización de estadísticas detalladas y debilidades de cada Pokémon.
* Interfaz minimalista inspirada en el diseño clásico de una Pokédex.
